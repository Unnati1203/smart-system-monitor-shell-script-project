# 🧠 Smart System Monitor (Shell Script)

## 📌 Overview
A lightweight shell script tool that monitors system performance and performs automatic corrective actions when thresholds are exceeded.

## ⚡ Features
- Monitors CPU, Memory, Disk usage
- Detects abnormal spikes
- Performs auto-healing actions
- Logs system activity

## 🛠 Tech Stack
- Bash (Shell Scripting)
- Linux utilities (top, free, df, ps)

## 🚀 Setup Instructions
```bash
git clone <your-repo-link>
cd smart-system-monitor
chmod +x install.sh
./install.sh
